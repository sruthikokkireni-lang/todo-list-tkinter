# To-Do List (Tkinter GUI)

A simple To-Do List application developed using Python and Tkinter.

## Features
- Add new tasks
- Delete completed tasks
- Easy to use GUI
- Lightweight application

## Technologies Used
- Python
- Tkinter

## How to Run

1. Install Python.
2. Save the file as `todo.py`.
3. Open Terminal or Command Prompt.
4. Run:

```bash
python todo.py
```
